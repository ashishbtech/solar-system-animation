![Project Preview](./assets/preview.gif)

## Live Demo
Check out the live project here: [https://ashishbtech.github.io/solar-system-animation](https://ashishbtech.github.io/solar-system-animation)

## Why I built this
Just a fun side project I made out of pure curiosity. I wanted to mess around with CSS keyframe animations, infinite orbits, and hover effects without relying on heavy libraries. The stars in the background are generated using a tiny bit of JavaScript just to give it that deep-space vibe. 

Made with code and lots of music.
